# Intravenous Anaesthetics

## Definition

> Intravenous anaesthetics are drugs that produce **loss of consciousness when administered intravenously**, usually within **one arm–brain circulation time (10–20 seconds)**.
> They are mainly used for **induction of general anaesthesia**, followed by maintenance with inhalational agents.

## Classification

| Class                                         | Examples                                 |
| :-------------------------------------------- | :--------------------------------------- |
| **Barbiturates**                              | `thiopental sodium`, `methohexital`      |
| **Dissociative anaesthetics**                 | `ketamine`                               |
| **Arylcyclohexylamines**                      | `ketamine`, `esketamine`                 |
| **Imidazole derivatives**                     | `etomidate`                              |
| **Alkylphenols**                              | `propofol`                               |
| **Benzodiazepines**                           | `midazolam`, `diazepam`, `lorazepam`     |
| **Opioid analgesics** _(used as supplements)_ | `fentanyl`, `alfentanil`, `remifentanil` |

## Mechanism of Action

Most IV anaesthetics act by **enhancing inhibitory neurotransmission** or **reducing excitatory activity** in the CNS:

| Class                                                              | Primary Action                                                                    |
| :----------------------------------------------------------------- | :-------------------------------------------------------------------------------- |
| **Barbiturates**, **benzodiazepines**, **propofol**, **etomidate** | Potentiate _GABA-A receptor_ activity → ↑ Cl⁻ influx → neuronal hyperpolarization |
| **Ketamine**                                                       | _NMDA receptor antagonist_ → inhibits excitatory glutamate transmission           |
| **Opioids**                                                        | _μ-opioid receptor agonists_ → analgesia and sedation                             |

## Pharmacological Features

| Feature         | Description                                                      |
| :-------------- | :--------------------------------------------------------------- |
| **Onset**       | Rapid (within 10–30 sec)                                         |
| **Duration**    | Short (5–10 min) due to redistribution from brain → muscle → fat |
| **Elimination** | Mainly hepatic metabolism, renal excretion                       |
| **Use**         | Induction of anaesthesia, short procedures, sedation in ICU      |

## Advantages

- Rapid induction and recovery
- Pleasant (no airway irritation)
- Useful in day-case and short procedures
- Can avoid stage of excitement seen in inhalational induction

## Disadvantages

- Requires IV access
- Difficult to control depth once injected
- Risk of overdose or apnea if given too rapidly
- Usually **not sufficient for maintenance alone** (except `propofol` infusion)

## Individual Drugs

### 1. `Thiopental sodium` (Barbiturate)

| Feature              | Description                                                                |
| :------------------- | :------------------------------------------------------------------------- |
| **Type**             | Ultra–short-acting barbiturate                                             |
| **Onset / Duration** | 15–30 sec / 5–10 min                                                       |
| **Mechanism**        | Enhances _GABA-A_ mediated inhibition                                      |
| **Uses**             | Induction of anaesthesia, control of status epilepticus, raised ICP        |
| **Adverse effects**  | Respiratory depression, hypotension, laryngospasm, porphyria precipitation |
| **Remarks**          | Redistributes rapidly → short action; **no analgesia**                     |

### 2. `Propofol`

| Feature              | Description                                                                                   |
| :------------------- | :-------------------------------------------------------------------------------------------- |
| **Type**             | Alkylphenol derivative                                                                        |
| **Onset / Duration** | 20 sec / 5–8 min                                                                              |
| **Mechanism**        | Potentiates _GABA-A_ action                                                                   |
| **Uses**             | Induction and maintenance (infusion), day-case surgery, sedation (ICU)                        |
| **Adverse effects**  | Hypotension, respiratory depression, pain on injection, **propofol infusion syndrome** (rare) |
| **Remarks**          | Smooth, rapid recovery; **antiemetic**, **no hangover**; _not analgesic_                      |

### 3. `Etomidate`

| Feature              | Description                                                    |
| :------------------- | :------------------------------------------------------------- |
| **Type**             | Imidazole derivative                                           |
| **Onset / Duration** | 15 sec / 5–10 min                                              |
| **Mechanism**        | _GABA-A receptor agonist_                                      |
| **Uses**             | Induction in patients with _cardiovascular instability_        |
| **Adverse effects**  | Pain on injection, myoclonus, adrenal suppression (↓ cortisol) |
| **Remarks**          | **Minimal CVS depression**; no analgesia                       |

### 4. `Ketamine`

| Feature              | Description                                                                                                       |
| :------------------- | :---------------------------------------------------------------------------------------------------------------- |
| **Type**             | Dissociative anaesthetic (arylcyclohexylamine)                                                                    |
| **Onset / Duration** | 30 sec / 10–15 min                                                                                                |
| **Mechanism**        | _NMDA receptor antagonist_ → inhibits excitatory glutamate transmission                                           |
| **Uses**             | Induction in shock, trauma, paediatric surgery, burn dressing                                                     |
| **Adverse effects**  | ↑ BP, ↑ HR (sympathomimetic), emergence delirium, hallucinations                                                  |
| **Remarks**          | Provides **analgesia + amnesia**; **maintains reflexes and respiration**; _useful in asthmatics_ (bronchodilator) |

### 5. `Midazolam` (Benzodiazepine)

| Feature              | Description                                                                    |
| :------------------- | :----------------------------------------------------------------------------- |
| **Type**             | Benzodiazepine sedative-hypnotic                                               |
| **Onset / Duration** | 2–3 min / 30–60 min                                                            |
| **Mechanism**        | Potentiates _GABA-A_ inhibition                                                |
| **Uses**             | Preoperative sedation, induction (with opioids), endoscopy procedures          |
| **Adverse effects**  | Respiratory depression (with opioids), hypotension                             |
| **Remarks**          | **Anxiolytic**, **amnestic**, **anticonvulsant**; reversible with `flumazenil` |

### 6. `Fentanyl` and related opioids

| Feature              | Description                                                            |
| :------------------- | :--------------------------------------------------------------------- |
| **Type**             | Potent _μ-opioid agonists_                                             |
| **Onset / Duration** | Rapid / short                                                          |
| **Uses**             | Supplement to GA, analgesia during induction and maintenance           |
| **Adverse effects**  | Respiratory depression, chest wall rigidity, nausea                    |
| **Remarks**          | Provides **analgesia**, not hypnosis; used in **balanced anaesthesia** |

## Comparative Summary

| Feature                   | `Thiopental` | `Propofol`        | `Etomidate`         | `Ketamine`         | `Midazolam`            |
| :------------------------ | :----------- | :---------------- | :------------------ | :----------------- | :--------------------- |
| **Onset (sec)**           | 20           | 20                | 15                  | 30                 | 120                    |
| **Duration (min)**        | 5–10         | 5–8               | 5–10                | 10–15              | 30–60                  |
| **Analgesia**             | None         | None              | None                | Excellent          | Moderate               |
| **Cardiovascular effect** | ↓ BP, ↓ CO   | ↓ BP              | Minimal             | ↑ BP, ↑ HR         | Mild ↓ BP              |
| **Respiration**           | Depressed    | Depressed         | Slight ↓            | Maintained         | Depressed              |
| **Special use**           | Raised ICP   | Day-case surgery  | Shock/CVS risk      | Asthma, shock      | Sedation               |
| **Adverse effects**       | Porphyria    | Pain on injection | Adrenal suppression | Emergence delirium | Respiratory depression |
