# Inhalational Anaesthetics

## Definition

> Inhalational anaesthetics are **volatile liquids or gases** that produce **general anaesthesia** when inhaled, acting primarily on the **central nervous system (CNS)** to induce **unconsciousness, amnesia, analgesia, and muscle relaxation**.

## Classification

| Type                             | Examples                                                            |
| :------------------------------- | :------------------------------------------------------------------ |
| **Gaseous anaesthetics**         | `nitrous oxide`, `xenon`                                            |
| **Volatile liquid anaesthetics** | `halothane`, `enflurane`, `isoflurane`, `sevoflurane`, `desflurane` |

## Ideal Properties of an Inhalational Anaesthetic

| Feature                  | Desirable Characteristic                        |
| :----------------------- | :---------------------------------------------- |
| **Induction & recovery** | Rapid and smooth                                |
| **Analgesia**            | Adequate pain relief during and after operation |
| **Muscle relaxation**    | Good skeletal muscle relaxation                 |
| **Safety margin**        | Wide therapeutic index                          |
| **Non-irritant**         | Pleasant odour, non-pungent                     |
| **Stability**            | Chemically stable, non-flammable                |
| **Minimal toxicity**     | Non-toxic to heart, liver, kidney               |
| **Controllability**      | Easy to adjust depth of anaesthesia             |
| **Compatibility**        | Compatible with other anaesthetic agents        |

## Mechanism of Action

- Inhalational agents **potentiate inhibitory neurotransmission** (mainly via _GABA-A_ and _glycine_ receptors).
- They **inhibit excitatory transmission** (e.g. _NMDA_, _nicotinic_ receptors).
- Result: _global CNS depression_ leading to **loss of consciousness, amnesia, analgesia, and immobility**.

## Important Concepts

| Term                                     | Meaning / Significance                                                                                                              |
| :--------------------------------------- | :---------------------------------------------------------------------------------------------------------------------------------- |
| **Minimum Alveolar Concentration (MAC)** | Concentration in alveoli that prevents movement in 50% of patients in response to surgical incision. _(Lower MAC = higher potency)_ |
| **Blood-gas partition coefficient**      | Solubility of drug in blood. _(Low coefficient = faster induction & recovery)_                                                      |
| **Oil-gas partition coefficient**        | Lipid solubility. _(Higher = more potent)_                                                                                          |

## Determinants of Induction and Recovery

| Factor                                                    | Effect                                                     |
| :-------------------------------------------------------- | :--------------------------------------------------------- |
| **Solubility in blood (blood–gas partition coefficient)** | Low solubility → rapid induction & recovery                |
| **Alveolar ventilation rate**                             | Higher ventilation → faster induction                      |
| **Cardiac output**                                        | Increased CO → slower induction (due to dilution in blood) |
| **Concentration of anaesthetic**                          | Higher concentration → faster induction                    |
| **Pulmonary blood flow**                                  | Uneven flow slows induction                                |

## Uptake and Distribution

1. **Inhalation → alveoli → blood → brain (site of action)**
2. Partial pressure gradient drives movement.
3. When inspired concentration equals alveolar concentration, **steady state** is reached.
4. Termination occurs by **exhalation and redistribution** to less perfused tissues (fat, muscle).

## Adverse Effects (General for Inhalational Agents)

| System                     | Effect                                                                                                                                  |
| :------------------------- | :-------------------------------------------------------------------------------------------------------------------------------------- |
| **Cardiovascular**         | Hypotension (vasodilation, myocardial depression)                                                                                       |
| **Respiratory**            | Respiratory depression, ↓ tidal volume, hypoxia                                                                                         |
| **CNS**                    | Increased intracranial pressure (especially `halothane`)                                                                                |
| **Liver**                  | _Halothane hepatitis_ (rare, idiosyncratic)                                                                                             |
| **Kidney**                 | _Enflurane_ → fluoride ion nephrotoxicity                                                                                               |
| **Malignant hyperthermia** | Rare, genetic; triggered by volatile agents + `succinylcholine` → muscle rigidity, ↑ temperature, acidosis. **Treatment:** `dantrolene` |
| **Uterine muscle**         | Relaxation → ↑ postpartum haemorrhage risk (especially `halothane`)                                                                     |

## Clinical Uses

| Use                                | Examples                                  |
| :--------------------------------- | :---------------------------------------- |
| **Induction & maintenance of GA**  | All inhalational agents                   |
| **Dental & minor surgery**         | `nitrous oxide`                           |
| **Paediatric anaesthesia**         | `sevoflurane` (non-irritant, rapid)       |
| **Maintenance after IV induction** | `isoflurane`, `desflurane`, `sevoflurane` |

## Advantages

- Easy depth control
- Rapid elimination through lungs
- Useful for long procedures
- Good muscle relaxation

## Disadvantages

- Expensive equipment required (vaporizer, scavenging system)
- Environmental pollution
- Risk of malignant hyperthermia
- Some agents are flammable or cause organ toxicity
