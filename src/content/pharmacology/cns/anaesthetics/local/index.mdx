# Local Anaesthetics

## Definition

> Local anaesthetics are drugs that produce **reversible loss of sensation in a limited area** of the body **without loss of consciousness**. They act by **blocking nerve impulse conduction** when applied locally to nerve tissue.

## Classification

### 1. Based on Chemical Linkage

| Class            | Examples                                                                            |
| :--------------- | :---------------------------------------------------------------------------------- |
| **Ester-linked** | `procaine`, `benzocaine`, `tetracaine`, `chloroprocaine`                            |
| **Amide-linked** | `lidocaine` (lignocaine), `bupivacaine`, `prilocaine`, `mepivacaine`, `ropivacaine` |

### 2. Based on Duration of Action

| Type                    | Examples                                   |
| :---------------------- | :----------------------------------------- |
| **Short-acting**        | `procaine`, `chloroprocaine`               |
| **Intermediate-acting** | `lidocaine`, `prilocaine`                  |
| **Long-acting**         | `bupivacaine`, `ropivacaine`, `tetracaine` |

### 3. Based on Site or Method of Application

| Method                 | Examples                     |
| :--------------------- | :--------------------------- |
| **Surface or topical** | `lidocaine`, `benzocaine`    |
| **Infiltration**       | `lidocaine`, `procaine`      |
| **Nerve block**        | `bupivacaine`, `mepivacaine` |
| **Spinal or epidural** | `bupivacaine`, `lidocaine`   |
| **Field block**        | `procaine`, `lidocaine`      |

## Mechanism of Action

- Local anaesthetics **block voltage-gated Na⁺ channels** in the neuronal membrane.
- This **prevents depolarization** and **halts propagation of action potentials**.
- Conduction block is **reversible** and **dose-dependent**.

**Sequence of loss of function:**
Pain → Temperature → Touch → Pressure → Motor function.
(_Small myelinated fibres are blocked earlier than large fibres._)

## Chemical Structure

Local anaesthetics share a basic structure with three parts:

1. **Lipophilic aromatic ring**
2. **Intermediate chain** – either **ester** or **amide linkage**
3. **Hydrophilic tertiary amine group**

The **intermediate chain** determines whether the drug is **ester** or **amide** type.

## Factors Affecting Local Anaesthetic Action

| Factor                                         | Effect                                                                     |
| :--------------------------------------------- | :------------------------------------------------------------------------- |
| **pH of tissue**                               | Acidic tissue (infection) → more ionized form → ↓ penetration → ↓ efficacy |
| **Concentration & dose**                       | ↑ Concentration → faster onset, deeper block                               |
| **Fibre diameter**                             | Small fibres blocked before large fibres                                   |
| **Use of vasoconstrictor (e.g. `adrenaline`)** | ↓ Systemic absorption → prolonged action, ↓ toxicity                       |

## Methods of Administration

| Method                       | Description                                                           |
| :--------------------------- | :-------------------------------------------------------------------- |
| **Surface anaesthesia**      | Applied to mucous membranes (eye, nose, mouth, throat, urethra)       |
| **Infiltration anaesthesia** | Injected around terminal nerve endings; minor surgical use            |
| **Field block anaesthesia**  | Injection around field; blocks sensory nerves before they enter area  |
| **Nerve block anaesthesia**  | Injection near a specific nerve or plexus                             |
| **Spinal anaesthesia**       | Injection into subarachnoid space (e.g. `bupivacaine`, `lidocaine`)   |
| **Epidural anaesthesia**     | Injection into epidural space                                         |
| **Topical anaesthesia**      | Direct application to skin or mucosa (e.g. `benzocaine`, `lidocaine`) |

## Adverse Effects

### 1. Central Nervous System

- Early: circumoral numbness, tinnitus, dizziness
- Later: convulsions, CNS depression, coma

### 2. Cardiovascular System

- Hypotension, bradycardia, cardiac arrest (especially with `bupivacaine`)

### 3. Allergic Reactions

- Common with **ester-type** drugs (due to _para-aminobenzoic acid_ formation)

### 4. Local Tissue Damage

- Repeated injection → neurotoxicity or myotoxicity

## Management of Toxicity

| Measure                   | Action                                                                |
| :------------------------ | :-------------------------------------------------------------------- |
| **Stop administration**   | Immediately discontinue LA use                                        |
| **Airway & oxygen**       | Maintain airway, give 100 % O₂                                        |
| **Control convulsions**   | `diazepam` or another benzodiazepine                                  |
| **Treat hypotension**     | Fluids, vasopressors                                                  |
| **Severe cardiotoxicity** | **IV lipid emulsion therapy** (especially for `bupivacaine` toxicity) |

## Uses

- Minor surgical and dental procedures
- Spinal and epidural anaesthesia in obstetric or abdominal operations
- Local infiltration for wound repair or biopsies
- Nerve blocks for pain relief
- Topical use in eye, ENT, and skin procedures
- _Antiarrhythmic use_: `lidocaine` (Class Ib antiarrhythmic)

### Important Examples

| Drug          | Type  | Onset    | Duration     | Special Points                        |
| :------------ | :---- | :------- | :----------- | :------------------------------------ |
| `lidocaine`   | Amide | Rapid    | Intermediate | Most widely used; low toxicity        |
| `bupivacaine` | Amide | Moderate | Long         | Used for spinal/epidural; cardiotoxic |
| `procaine`    | Ester | Slow     | Short        | Prototype; rarely used now            |
| `benzocaine`  | Ester | Rapid    | Short        | Topical only; water-insoluble         |
| `prilocaine`  | Amide | Moderate | Intermediate | Used with `lidocaine` in _EMLA cream_ |
