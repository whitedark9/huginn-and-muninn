# Nitrates

## Examples

| Type                | Examples                                   | Notes                                         |
| :------------------ | :----------------------------------------- | :-------------------------------------------- |
| **Short-acting**    | `nitroglycerin` (glyceryl trinitrate, GTN) | Sublingual/spray — rapid relief               |
| **Intermediate**    | `isosorbide dinitrate`                     | Oral/sublingual — moderate duration           |
| **Long-acting**     | `isosorbide mononitrate`                   | Oral, sustained release                       |
| **Parenteral / IV** | `nitroglycerin`, `sodium nitroprusside`    | Hypertensive emergencies, acute heart failure |

> **Venodilation → ↓ preload** (↓ venous return).<br/>
> **Arteriolar dilation → ↓ afterload** (↓ peripheral resistance).<br/>
> **Coronary vasodilation → ↑ O₂ supply**.

## Mechanism of Action (MOA)

- **Nitrates** are **prodrugs** → release **nitric oxide (NO)** in vascular smooth muscle.
- NO activates **guanylyl cyclase**, ↑ **cGMP** → activation of **protein kinase G (PKG)** →
  - Dephosphorylation of myosin light chain → **smooth muscle relaxation** → **vasodilation**.

## Pharmacodynamics

| Site                      | Effect                                | Clinical Result               |
| :------------------------ | :------------------------------------ | :---------------------------- |
| **Venous system (major)** | ↑ venous capacitance → ↓ preload      | ↓ myocardial O₂ demand        |
| **Arterioles (moderate)** | ↓ afterload                           | ↓ wall stress                 |
| **Coronary arteries**     | Dilatation, especially of collaterals | ↑ perfusion to ischemic areas |
| **Cerebral vessels**      | Vasodilation                          | Headache (common side effect) |

- **Reflex tachycardia** may occur (due to baroreceptor activation).

## Pharmacokinetics

| Drug                     | Route      |   Onset   |      Duration       | Notes                |
| :----------------------- | :--------- | :-------: | :-----------------: | :------------------- |
| `nitroglycerin`          | Sublingual |  1–3 min  |       30 min        | For acute angina     |
| `isosorbide dinitrate`   | Oral       | 15–30 min |        4–6 h        | Active metabolites   |
| `isosorbide mononitrate` | Oral (SR)  | 30–60 min |       8–10 h        | High bioavailability |
| `nitroprusside`          | IV         | Immediate | 1–2 min (stop→stop) | For emergencies      |

- **Metabolism:** Hepatic (extensive first-pass for oral GTN).
- **Elimination:** Mainly renal.

## Indications

- **Angina pectoris** (acute & prophylactic)
- **Chronic heart failure** (reduce preload)
- **Hypertensive emergencies** (`nitroprusside`, IV GTN)
- **Acute pulmonary edema** (↓ venous return, ↓ LV filling pressure)

## Adverse Effects

| System                           | Effect                                         | Mechanism                      |
| :------------------------------- | :--------------------------------------------- | :----------------------------- |
| **CVS**                          | Postural hypotension, reflex tachycardia       | Vasodilation                   |
| **CNS**                          | Headache, flushing, dizziness                  | Cerebral vasodilation          |
| **Skin**                         | Flushing                                       | —                              |
| **Metabolic (IV nitroprusside)** | **Cyanide toxicity** (esp. prolonged infusion) | Release of CN⁻ ions            |
| **Tolerance**                    | ↓ effect on continuous exposure                | Depletion of sulfhydryl groups |

## Cautions

- Avoid **continuous use** → tolerance develops.
  - Use **“nitrate-free interval”** (e.g. 8–12 h overnight).

- Caution in **hypotension**, **volume depletion**, or **raised ICP**.

## Contraindications

- **Concurrent PDE-5 inhibitors (sildenafil, tadalafil, vardenafil)** → **severe hypotension**.
- **Severe anemia** (↓ O₂ delivery)
- **Hypotension / shock states**
- **Increased intracranial pressure**
- **Hypertrophic obstructive cardiomyopathy (HOCM)** (may worsen obstruction)

## Drug Interactions

- **PDE-5 inhibitors** → life-threatening hypotension.
- **Alcohol** → enhances vasodilation.
- **Other antihypertensives** → additive hypotensive effect.

## Key Point Summary

| Feature                  | **Nitrates**                                                    |
| :----------------------- | :-------------------------------------------------------------- |
| **Mechanism**            | NO release → ↑ cGMP → vasodilation                              |
| **Major Effect**         | Venodilation > arteriolar dilation                              |
| **Main Uses**            | Angina, HF, hypertensive emergency                              |
| **Adverse Effects**      | Headache, flushing, hypotension, reflex tachycardia             |
| **Tolerance**            | Common — use nitrate-free interval                              |
| **Contraindicated With** | PDE-5 inhibitors (e.g. sildenafil)                              |
| **Special Drug**         | `nitroprusside` → rapid IV BP control, risk of cyanide toxicity |

## ⚡️ **Nitrates vs Nitroprusside — Comparison**

| Feature              | **Organic Nitrates (GTN, ISDN, ISMN)** | **Sodium Nitroprusside**                   |
| :------------------- | :------------------------------------- | :----------------------------------------- |
| **Mechanism**        | NO donor → ↑ cGMP                      | Direct NO donor → ↑ cGMP                   |
| **Vessel Target**    | Mainly veins (↓ preload)               | Arterioles & veins (↓ preload + afterload) |
| **Onset / Duration** | Rapid (sublingual GTN: 1–3 min)        | Immediate onset, short duration            |
| **Use**              | Angina, HF                             | Hypertensive emergency                     |
| **Major Risk**       | Tolerance                              | Cyanide toxicity                           |
| **Bradykinin**       | No effect                              | No effect                                  |
